# Road Accident Analysis Dashboard

## Project Title / Headline
🚗 Road Accident Analysis Dashboard: Traffic Accident Insights using Power BI

An interactive Power BI dashboard designed to analyze road accident data, monitor casualty trends, identify high-risk areas, and support data-driven decisions for improving road safety.

---
## Short Description / Purpose

The Road Accident Analysis Dashboard provides a comprehensive overview of road accident statistics, including casualties, accident severity, vehicle types, road conditions, urban vs rural distribution, and monthly trends. The dashboard enables transportation authorities, policymakers, and safety analysts to identify accident patterns and implement effective road safety measures.

---
## Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 Power BI Desktop – Dashboard development and interactive visualization.
- 🔄 Power Query – Data cleaning, transformation, and preprocessing.
- 📐 DAX (Data Analysis Expressions) – Calculated measures, KPIs, and dynamic insights.
- 🗂️ Data Modeling – Establishing relationships between accident-related datasets.
- 📄 Excel/CSV Dataset – Source of road accident records.
- 🎨 Power BI Visualizations – KPI Cards, Line Charts, Bar Charts, Donut Charts, Slicers, Tables, and Interactive Filters.

---

## Data Source

- UK Road Accident Dataset (Excel/CSV)
- Dataset includes information such as:
  - Accident Date
  - Casualties
  - Accident Severity
  - Vehicle Type
  - Road Type
  - Light Condition
  - Urban/Rural Area
  - Local Authority
  - Month & Year
  - Accident Location

## Business Insights

- Identifies accident-prone locations with the highest casualty counts.
- Compares current year accident performance with the previous year.
- Highlights the percentage of fatal, serious, and slight casualties.
- Analyzes the impact of vehicle types on accident severity.
- Evaluates accident distribution across urban and rural areas.
- Examines the influence of daylight and darkness on road accidents.
- Supports transportation authorities in improving road safety and planning preventive measures.
